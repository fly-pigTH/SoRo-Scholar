**SoRo-Scholar** is a community-curated map of soft robotics research -- papers, researchers, labs, conferences, and tools.

Started by [Yinglei Zhu](https://fly-pigth.github.io/) and [Qiyao Wang](https://king-bridge.github.io/) in October 2025.

---

## Papers

Organized by topic. Each category links to a dedicated page with year-by-year listings.

| Topic | Highlights |
|---|---|
| **[Reviews & Surveys](papers/reviews.md)** | Rus & Tolley (Nature 2015), Whitesides (Angew. Chem. 2018) |
| **[Actuation](papers/actuation.md)** | Multigait soft robot (PNAS 2011), HASEL actuators (Science 2018) |
| **[Sensing](papers/sensing.md)** | Stretchable electroluminescent skin (Science 2016) |
| **[Fabrication](papers/fabrication.md)** | 3D printing of soft robotic systems (Nature Rev. Mater. 2018) |
| **[Bio-Inspired](papers/bio-inspired.md)** | Mariana Trench soft robot (Nature 2021), Robotic fish (Sci. Rob. 2018) |
| **[Medical](papers/medical.md)** | Biomedical soft robotics (Nature Rev. Mater. 2018) |
| **[Human-Robot Interaction](papers/hri.md)** | Soft wearable robots (Science 2017) |
